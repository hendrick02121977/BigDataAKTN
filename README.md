## 📚 Course Description

This repository contains a comprehensive **Big Data Analytics** course designed for magister (master's) students. All materials are provided as interactive Jupyter notebooks that run directly in **Google Colab** — no local installation required.

Students will learn the full Big Data pipeline: from foundational concepts and distributed storage through data processing, machine learning, and real-world case studies, using industry-standard tools including Python, PySpark, and cloud-native services.

## 🚀 Getting Started

### Using Google Colab (Recommended)
1. Click any **"Open in Colab"** badge above.
2. Sign in with your Google account.
3. Choose **Runtime → Run all** to execute the notebook.
4. Save a copy to your own Google Drive via **File → Save a copy in Drive**.

### Running Locally
```bash
git clone https://github.com/hendrick02121977/BigDataAKTN.git
cd BigDataAKTN
pip install jupyter notebook pandas numpy matplotlib seaborn scikit-learn plotly pyspark
jupyter notebook
```

## 🎓 Learning Outcomes

Upon completing this course, students will be able to:
- Explain key Big Data concepts, architectures, and the Hadoop ecosystem
- Ingest, clean, transform, and analyse large datasets with Python
- Build and evaluate machine learning models on structured and unstructured data
- Execute distributed data processing jobs with Apache Spark
- Design appropriate storage solutions (relational, NoSQL, Data Lake)
- Deliver a complete end-to-end Big Data analytics project
